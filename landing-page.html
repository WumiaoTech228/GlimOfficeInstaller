开始使用 GOI

项目基于 .NET Framework 4.8，支持 Visual Studio 2022 或 dotnet CLI 构建。

  • 1 克隆仓库 git clone
  • 2 使用 dotnet 构建 Release 版本
  • 3 以管理员身份运行 GOI.exe

需要管理员权限（涉及注册表修改与系统服务操作）。

# 克隆仓库 git clone https://github.com/WumiaoTech228/GlimOfficeInstaller.git # 构建 Release dotnet build -c Release # 编译完成！产物路径： bin/Release/net48/GOI.exe
