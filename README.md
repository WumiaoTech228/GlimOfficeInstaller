# GlimOfficeInstaller
一款便携，轻量化，原版化的微软Office安装软件
众所周知，一款优秀的开源程序是不需要逻辑严密的markdown的
本软件目前最主要的功能是安装并激活Office，可以由用户自行选择要安装的Office组件，并且能在安装之后用Ohook功能进行激活。还有点神秘小功能：点击Banner部分五次可打开隐藏菜单，里面有激活Windows、Office、WindowsESU功能，查看Windows、Office激活状态，更改Office授权信息（目前该功能不成熟）。点击主页面的“产品选择”字样可以查看本软件信息。点击五次隐藏功能菜单中的“隐藏功能菜单”字样可打开软件日志查看功能。点击“来都来了”菜单中“来都来了，选一个吧”字样五次，可打开ESU激活菜单。
请注意：本软件产生的日志以及部署所下载的文件会放置在本软件可执行文件（后缀名为.exe）的文件夹下，请勿随便删除，若自己提前下载好ODT的可执行下载文件和configuration.xml可以加速配置直接下载并安装Office的文件。
